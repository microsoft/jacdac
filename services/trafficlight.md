# Traffic Light

    identifier: 0x15c38d9b
    tags: 8bit

Controls a mini traffic with a red, orange and green LED.

## Registers

    rw red: bool @ 0x80

The on/off state of the red light.

    rw orange: bool @ 0x81

The on/off state of the red light.

    rw green: bool @ 0x82

The on/off state of the red light.
